<h1 align="center">EX – TypeScript Rifai un Esercizio</h1>

Esercitazione sviluppata in **TypeScript** che consiste nella **riscrittura di un esercizio sviluppato in JavaScript**, **“Il compleanno dello chef”**.

L’obiettivo è mantenere lo stesso comportamento della versione originale, applicando i concetti fondamentali di TypeScript come la tipizzazione delle funzioni, delle variabili e dei valori restituiti dalle Promise.
